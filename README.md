# bikeapp

## Carregando o backend:
$ node www/load.js

## Iniciando a aplicação:
$ node www/start.js

## Configurando a url do app:
www/js/applicationService.js
apiPath: 'http://192.168.0.10:3200/api',

## Executando o app
$ ionic serve
